# Diabetes Prediction ML Model

This project is a Machine Learning model that predicts whether a person has diabetes or not based on medical attributes. The model is built using Python and popular ML libraries.

## 🚀 Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Diabetes prediction system
- Simple and beginner-friendly project

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📂 Project Structure

├── diabetes.csv  
├── diabetes_prediction.ipynb  
├── trained_model.sav  
└── README.md  

## 📊 Dataset

The dataset used in this project contains medical information such as:

- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## 🤖 Machine Learning Model

The project uses a Machine Learning classification algorithm to predict diabetes based on input data.

## 📈 Workflow

1. Import Dataset
2. Data Cleaning
3. Data Analysis
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Prediction System
